Get BloxChat installed and start chatting in Roblox games in just a few minutes.

<Steps>
  <Step title="Download and install BloxChat">
    Download the latest installer from the [GitHub Releases](https://github.com/logixism/bloxchat/releases/latest) page.

    <Note>
      BloxChat requires **Windows 10** or **Windows 11**. WebView2 is required and will be automatically installed if missing.
    </Note>

    Run the `.msi` installer and follow the installation wizard. BloxChat will be installed to:
    ```
    C:\Users\<YourUsername>\AppData\Local\Programs\BloxChat
    ```

    Launch BloxChat from your Start menu or desktop shortcut.
  </Step>

  <Step title="Complete Roblox verification">
    When you first launch BloxChat, you'll need to verify your Roblox account.

    <Info>
      Verification links your Roblox account to BloxChat securely without requiring your password.
    </Info>

    **Verification steps:**

    1. Click **Start Verification** in the BloxChat login window
    2. You'll receive a **6-digit verification code**
    3. Join the verification place in Roblox (link provided in the app)
    4. The game will automatically detect and verify your code
    5. Return to BloxChat — you'll be logged in automatically

    <Warning>
      The verification code expires in **10 minutes**. If it expires, click **Start Verification** again to get a new code.
    </Warning>

    Your session lasts for **1 hour**. After that, BloxChat will refresh your session automatically in the background.
  </Step>

  <Step title="Join a Roblox game and start chatting">
    Once verified, you're ready to chat!

    1. **Join any Roblox game** that uses BloxChat
    2. BloxChat automatically detects the game server (**JobId**) from your Roblox logs
    3. You'll see a new channel appear in BloxChat for that server
    4. **Type your message** and press `Enter` to send

    <Tip>
      Press `/` to auto-focus the chat input. Press `Esc` to return focus to Roblox without alt-tabbing.
    </Tip>

    **Channel switching:**
    - Each Roblox server (JobId) gets its own chat channel
    - Switch servers in Roblox, and BloxChat updates automatically
    - No manual channel selection needed!
  </Step>
</Steps>

## Automatic behaviors

<AccordionGroup>
  <Accordion title="Channel switching" icon="arrows-rotate">
    BloxChat monitors your Roblox log file and automatically switches channels when you join a new server. You don't need to manually select channels.
    
    If channels aren't switching, verify your **Roblox logs path** in Settings.
  </Accordion>

  <Accordion title="Auto-updates" icon="download">
    On Windows, BloxChat checks for updates every time it starts. If a new version is available, it downloads and installs automatically.
    
    You'll see a notification when an update is ready. Just restart the app to apply it.
  </Accordion>

  <Accordion title="Session refresh" icon="clock-rotate-left">
    Your JWT session expires after 1 hour, but BloxChat refreshes it automatically in the background. You won't be logged out unless the refresh fails.
    
    If you're logged out unexpectedly, simply verify again.
  </Accordion>
</AccordionGroup>

## Common first-time issues

<AccordionGroup>
  <Accordion title="BloxChat can't connect to the server" icon="wifi-slash">
    **Solution:** Check **Settings → API Server URL**. The default is `https://bloxchat.logix.lol`.
    
    If you're using a self-hosted backend, make sure to include `http://` or `https://` explicitly:
    ```
    http://localhost:3000
    ```
  </Accordion>

  <Accordion title="Verification code doesn't work" icon="circle-xmark">
    **Possible causes:**
    - Code expired (10-minute limit)
    - Wrong verification place
    - Backend server not configured correctly
    
    **Solution:** Start a new verification session and join the place immediately.
  </Accordion>

  <Accordion title="Channels don't switch when I change servers" icon="server">
    **Solution:** Verify your **Roblox logs path** in Settings. BloxChat needs read access to:
    ```
    C:\Users\<YourUsername>\AppData\Local\Roblox\logs
    ```
    
    If the path is wrong, BloxChat can't detect server changes.
  </Accordion>
</AccordionGroup>
